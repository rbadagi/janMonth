## 🌟 Problem Being Solved

**Find the length of the longest substring without repeating characters**.

**Input**

```python
s = "abcabcbb"
```

***

## 🔧 Code (for reference)

```python
s = "abcabcbb"
char_set = set()
left = 0
max_len = 0

for right in range(len(s)):
    while s[right] in char_set:
        char_set.remove(s[left])
        left += 1
    char_set.add(s[right])
    max_len = max(max_len, right - left + 1)

print(max_len)
```

***

## 🧠 Meaning of Variables

| Variable   | Purpose                                        |
| ---------- | ---------------------------------------------- |
| `left`     | Start index of window                          |
| `right`    | End index of window                            |
| `char_set` | Stores unique characters in the current window |
| `max_len`  | Length of the longest valid substring found    |

The window is always:

```text
s[left : right + 1]
```

***

## 🔁 Step-by-Step Execution

We’ll track:

*   `right`
*   Current character
*   `char_set`
*   `left`
*   `max_len`

***

### ✅ Step 1: `right = 0`

Character: `'a'`

*   `'a'` not in `char_set`
*   Add `'a'`

```text
Window: "a"
char_set = {'a'}
left = 0
max_len = max(0, 0 - 0 + 1) = 1
```

***

### ✅ Step 2: `right = 1`

Character: `'b'`

*   `'b'` not in `char_set`
*   Add `'b'`

```text
Window: "ab"
char_set = {'a', 'b'}
left = 0
max_len = max(1, 1 - 0 + 1) = 2
```

***

### ✅ Step 3: `right = 2`

Character: `'c'`

*   `'c'` not in `char_set`
*   Add `'c'`

```text

char_set = {'a', 'b', 'c'}
left = 0
max_len = max(2, 2 - 0 + 1) = 3 ✅
```

***

### ⚠️ Step 4: `right = 3`

Character: `'a'` (duplicate!)

*   `'a'` **is already in `char_set`**
*   We must **shrink the window from the left**

#### While loop begins:

    Remove s[left] → 'a'
    left = 1

Now `'a'` is removed.

*   Add `'a'` again

```text
Window: "bca"
char_set = {'b', 'c', 'a'}
left = 1
max_len = max(3, 3 - 1 + 1) = 3
```

***

#### While loop:

Window: "cab"

**Longest substring(s):**

***

## 🧩 Key Insights

✅ `while` loop ensures **no duplicates**  
✅ Window expands with `right`  
✅ Window shrinks with `left`  
✅ Time complexity: **O(n)**  
✅ Space complexity: **O(n)**
