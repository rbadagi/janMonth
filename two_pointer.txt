## ✅ What is the Two Pointer Pattern?

The **Two Pointer Pattern** uses **two indices (pointers)** to traverse a data structure (usually an array or string).  
It helps solve problems **efficiently**—often reducing time complexity from **O(n²) to O(n)**.

***

## ✅ When to Use It

*   Sorted arrays
*   Pair-sum problems
*   Reversing sequences
*   Removing duplicates
*   Finding subarrays or pairs

***

## ✅ Common Two Pointer Types

### 1️⃣ Opposite Direction Pointers

*   One pointer starts at the **beginning**
*   Another starts at the **end**
*   Pointers move toward each other

### 2️⃣ Same Direction (Fast & Slow)

*   Both start at the beginning
*   One moves faster than the other

***

## ✅ Example 1: Check if Array Has Two Numbers with Target Sum

**(Sorted Array)**

### ✅ Python Program

```python
def two_sum(arr, target):
    left = 0
    right = len(arr) - 1

    while left < right:
        current_sum = arr[left] + arr[right]

        if current_sum == target:
            return True
        elif current_sum < target:
            left += 1
        else:
            right -= 1

    return False

arr = [1, 2, 3, 4, 6]
target = 6
print(two_sum(arr, target))
```

🟢 **Output**

    True

⏱ Time Complexity: **O(n)**  
📦 Space Complexity: **O(1)**

***

## ✅ Example 2: Reverse an Array (Two Pointers)

### ✅ Python Program

```python
def reverse_array(arr):
    left = 0
    right = len(arr) - 1

    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1

    return arr

print(reverse_array([1, 2, 3, 4, 5]))
```

🟢 **Output**

    [5, 4, 3, 2, 1]

**(Fast and Slow Pointer)**

### ✅ Python Program
