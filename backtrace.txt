## 1. What is Backtracking in DSA?

**Backtracking** is an algorithmic technique used to solve problems **incrementally**, where we build a solution step by step and **remove (backtrack)** those steps that do not lead to a valid solution.

👉 In simple words:

> **Try → Check → If fails → Undo and try another option**

Backtracking is essentially a smarter brute‑force approach that **avoids unnecessary computations**.

### Key Idea

*   Explore all possible choices recursively
*   Abandon a path as soon as it becomes invalid
*   Go back to the previous state and try the next choice

***

## 3. When Should You Use Backtracking?

Use backtracking when:

*   You must generate **all possible solutions**
*   The problem has **constraints**
*   Choices depend on **previous decisions**

### Common Characteristics:

*   Decision tree
*   Recursive calls
*   Undoing changes (backtrack step)

## 4. General Backtracking Template

```python
def backtrack(current_state):
    if solution_found(current_state):
        record_answer()
        return

    for choice in possible_choices:
        if is_valid(choice):
            make_choice(choice)
            backtrack(current_state)
            undo_choice(choice)
```

### 2️⃣ Subsets / Power Set

Generate all subsets of a given set.

**Example:**
Input: `[1,2]`  
Output: `[[], [1], [2], [1,2]]`

```python
def subsets(nums):
    result = []

    def backtrack(index, path):
        result.append(path[:])
        for i in range(index, len(nums)):
            path.append(nums[i])
            backtrack(i + 1, path)
            path.pop()

    backtrack(0, [])
    return result
```

***

## 7. Time & Space Complexity (Interview Insight)

*   **Time Complexity:** Usually **exponential** (O(2ⁿ), O(n!), etc.)
*   **Space Complexity:** O(depth of recursion)

👉 Interviewers expect you to **acknowledge exponential time** and explain pruning.

## 8. Backtracking vs Recursion

| Aspect      | Recursion     | Backtracking          |
| ----------- | ------------- | --------------------- |
| Purpose     | Solve problem | Explore all solutions |
| Undo step   | Not mandatory | Mandatory             |
| Constraints | Optional      | Core idea             |

✅ **All backtracking uses recursion, but not all recursion is backtracking**
